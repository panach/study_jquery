기본 형태

이형태는 for 와 id 를 기반으로 동작합니다. 그래서 inupt 과 label 의 순서에 관계없이 작동한다 label 을 클릭하면 배경이 교체되는 기본 형이다.
input 은 checked 속성이 엘리먼트에 추가/삭제되고
label 의 class 가 추가/삭제 된다.

구조

<input type="checkbox" id="ipt_ckb1" class="hide">
<label for="ipt_ckb1" class="ipt_ckb active">
    test 텍스트
</label>
