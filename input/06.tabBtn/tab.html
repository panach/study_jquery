탭 순환 버튼

좌우 버튼을 눌러 가운대 있는 div 를 순자적으로 display: none/block 시킨다
방법으로는 index 를 변수에 담고 해당 변수 기반으로 동작하게 하는것
지금 이해가 안된다고 하더라도 다 방법을 모두 만들어 보면 차이점을 느낄수 있을것이다.

Type1

현제 block 속성을 가지는 엘리먼트를 찾아 인덱스를 찾고 동작 시킨다 자세한 내용은 js 의 주석을 활용하자

구조

    <div class="tab_year">
        <div class="current_year">

        <!-- [D] display: none/block 으로 제어 가능합니다. -->
        <div class="y2015" style="display:block"><span class="blind">2015</span></div>
        <div class="y2014" style="display:none"><span class="blind">2014</span></div>
        <div class="y2013" style="display:none"><span class="blind">2013</span></div>
        </div>
        <a href="#" class="btn_year_prev"><span class="blind">이전 년도 보기</span><em class="blind">비활성화</em></a>
        <a href="#" class="btn_year_next"><span class="blind">다음 년도 보기</span><em class="blind">비활성화</em></a>
    </div>
