input 체크박스 커스텀 하기

기본 형태

label 을 클릭하면 배경이 교체되는 기본 형이다.
input 은 checked 속성이 엘리먼트에 추가/삭제되고
label 의 class 가 추가/삭제 된다.
이형태는 label 의 for 와 input 의 id 가 없어도 동작 하며
대신 input 과 label 의 순서에 의해 트래킹하고 작동된다

구조

<input type="checkbox" class="hide">
<label class="ipt_ckb active">
    test 텍스트
</label>
