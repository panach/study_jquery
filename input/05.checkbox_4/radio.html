라디오 버튼 만들기

Type1

사용 조건이 있다
lebel 내부에 input 과 텍스트가 같이 있어야 한다.
input의 id 값이 있으며 label 의 for 는 오류 없이 input 의 label 과 매칭이 되어야 한다.
그로 인해 label 을 클릭시 radio 의 기본동작이 작동해야 한다.
name 을 통한 고유 기능은 유지 되어야 한다.
for 와 name 의 기본기능을 마크업에서 제거 하고 js 로 구현하는 형태는 다른 타입으로 구현하도록 한다.

라디오 버튼의 활성화 class 명은 active 이다

구조

    <label for="ipt_rdo1">
        <input type="radio" name="ipt" id="ipt_rdo1" class="hide">
        test 텍스트
    </label>
